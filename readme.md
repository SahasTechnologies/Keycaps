#Keycaps

These are keycaps that I made for a general button, a tab button and a space bar!

<img width="890" height="746" alt="image" src="https://github.com/user-attachments/assets/8e3220e7-47cd-4e17-8e8b-02ed5d98d974" />

The button at the top is a tab button (the gray one), the middle one is a regulat keycap, while the one on the bottom is a space bar. All 3 of them use Lucide Icons for the icons on the top with a 0.5mm indentation so that it can be felt with a finger while typing. I specificalyl use lucide icons without any text to create minimalist icons that are not language based and anyone can print and stick to their keyboard even if they do not understand English.
